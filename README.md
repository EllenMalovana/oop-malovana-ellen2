# oop-malovana-ellen2

## Spuštění
1. Spusť lokální server (např. Live Server)
2. Otevři `index.html` v prohlížeči  
3. Otevři konzoli (F12) → uvidíš výstupy programu

## Obsah
- `Product.js` – třída Product
- `main.js` – hlavní skript s instancemi
- `index.html` – spouštěcí HTML soubor
- `console-screenshot.png` – screenshot výstupu
